О сервисе "РыбаТекст.рф"

Сервис "РыбаТекст.рф" генерирует заданное количество абзацев более-менее осмысленного текста на русском языке. В англоязычном варианте классическим примером такого рыба-текста является Lorem Ipsum.

Данный генератор текста будет полезен для верстальщиков и дизайнеров, которым требуется рандомный текст для заполнения сайтов и макетов.

Генерация текста осуществляется по количеству абзацев или по количеству символов.

Не стоит использовать сгенерированный текст для реальных проектов. Используйте рыбный текст только для тестирования и прототипирования. Для готового проекта лучше взять осмысленную статью.
